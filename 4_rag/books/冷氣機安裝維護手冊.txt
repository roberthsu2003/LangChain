第一章: 安裝說明

1.1 安裝前注意事項
- 請務必委由專業技術人員安裝
- 自行安裝可能造成危險且影響保固
- 安裝前確認電源規格是否符合
- 確認安裝位置符合規範

1.2 室內機安裝位置選擇
適合安裝位置:
- 牆面堅固,可承受室內機重量(15kg以上)
- 距離天花板至少15cm
- 距離側牆至少10cm
- 排水順暢處
- 空氣循環良好處
- 遙控器訊號可達處

避免安裝位置:
- 陽光直射處
- 熱源附近(廚房、暖爐)
- 油煙重的地方
- 易受水氣影響處(浴室)
- 會影響睡眠的位置(床頭正上方)
- 窗簾或傢俱會擋住出風口

1.3 室外機安裝位置選擇
適合安裝位置:
- 通風良好處
- 可承重50kg以上的平台或牆面
- 方便維修檢查處
- 排水便利處

避免安裝位置:
- 陽光直射處(建議加裝遮陽板)
- 會積水處
- 會被雨淋濕的位置
- 腐蝕性氣體環境
- 鄰居窗戶正對面(噪音考量)

安裝空間需求:
- 前方: 至少50cm
- 後方: 至少10cm
- 左右: 各至少20cm
- 上方: 至少50cm

1.4 冷媒管路配置
冷媒管長度:
- 標準配管: 3公尺(含)
- 最大配管: 15公尺
- 超過標準需額外收費

配管注意事項:
- 避免過度彎折
- 每3公尺需固定一次
- 穿牆處需使用護套
- 確實包覆保溫材
- 冷媒管不可破損

1.5 排水管安裝
- 排水管需保持1/100以上斜度
- 每50cm固定一次
- 出水口需高於地面或排水溝
- 不可與污水管直接連通
- 冬季需注意防凍

1.6 電源配線
電源規格:
- 110V機型: 專用迴路15A以上
- 220V機型: 專用迴路10A以上
- 務必使用接地型插座
- 電源線徑需符合規範

配線注意事項:
- 使用專用迴路(不與其他電器共用)
- 確實接地
- 不可使用延長線
- 配線符合屋內線路裝置規則

1.7 試運轉
安裝完成後請確認:
1. 冷媒壓力正常(專業人員檢測)
2. 排水測試正常
3. 各項功能正常運作
4. 無異常噪音或振動
5. 遙控器功能正常
6. 拿到完整保固文件

第二章: 遙控器操作

2.1 遙控器基本功能
按鍵說明:
- 開/關: 啟動或停止運轉
- 模式: 切換運轉模式(冷氣/除濕/送風/暖氣)
- 溫度+/-: 調整設定溫度(16-30°C)
- 風量: 切換風速(自動/微風/弱/中/強)
- 風向: 調整上下出風角度
- 擺動: 自動擺動出風葉片
- 省電: 啟動省電模式
- 靜音: 啟動靜音運轉
- 定時: 設定開/關機時間
- 睡眠: 啟動睡眠模式

顯示符號:
- 雪花: 冷氣模式
- 太陽: 暖氣模式
- 水滴: 除濕模式
- 風扇: 送風模式
- 溫度數字: 設定溫度
- 風量圖示: 當前風速

2.2 運轉模式說明

冷氣模式:
- 夏季降溫使用
- 可設定溫度16-30°C
- 建議設定26-28°C最省電
- 溫度每調高1°C可省電約6%

暖氣模式:
- 冬季供暖使用
- 可設定溫度16-30°C
- 建議設定20-22°C最舒適
- 室外溫度低於5°C效率降低

除濕模式:
- 梅雨季節或潮濕時使用
- 自動設定最佳溫度與風量
- 不可手動調整溫度
- 連續使用建議不超過4小時

送風模式:
- 僅循環空氣,不調溫
- 可當電風扇使用
- 最省電的模式
- 適合春秋季節

自動模式:
- 自動判斷運轉模式
- 根據室溫自動切換冷/暖氣
- 維持設定溫度±2°C範圍
- 最便利的選擇

2.3 進階功能

省電模式:
- 微幅調高/降低設定溫度
- 降低壓縮機運轉頻率
- 可省電約20-30%
- 舒適度略降

靜音模式:
- 降低運轉音量至最低
- 風量自動調整為微風
- 適合睡眠或工作時使用
- 冷房能力降低約30%

睡眠模式:
- 自動調整溫度(每小時±1°C)
- 避免過冷或過熱
- 減少噪音
- 可設定1-8小時後自動關機

定時功能:
- 可設定0.5-24小時
- 定時開機(回家前先開冷氣)
- 定時關機(睡眠時使用)
- 開關機可同時設定

風向控制:
- 5段上下角度調整
- 自動擺動功能
- 冷氣建議水平送風
- 暖氣建議往下送風

第三章: 清潔保養

3.1 濾網清潔(每2週)
清潔步驟:
1. 關閉電源開關
2. 打開室內機前面板
3. 取出濾網(往上提起)
4. 用吸塵器吸除灰塵
5. 用清水沖洗(水溫<40°C)
6. 用軟毛刷輕刷
7. 晾乾後裝回(務必完全乾燥)
8. 關閉前面板

注意事項:
- 不可用熱水(>40°C)
- 不可用刷子用力刷
- 不可用洗衣機洗
- 不可烘乾或曝曬
- 損壞需更換新品

3.2 外殼清潔(每月)
室內機外殼:
1. 關閉電源
2. 用乾布或微濕布擦拭
3. 頑固污漬用中性清潔劑
4. 不可用水直接沖洗
5. 擦拭後用乾布擦乾

室外機外殼:
1. 關閉電源
2. 清除雜物落葉
3. 用水沖洗鰭片(由上往下)
4. 不可用高壓水柱
5. 注意不要弄濕電氣零件

第四章: 故障排除

4.1 常見問題診斷

問題: 無法開機
可能原因:
1. 電源未接通 → 檢查插頭、開關、斷路器
2. 遙控器電池沒電 → 更換電池
3. 保護裝置啟動 → 等待3-5分鐘後重試
4. 電源電壓異常 → 檢查電源供應

問題: 不冷或冷房效果差
可能原因:
1. 濾網髒污 → 清洗濾網
2. 設定溫度過高 → 降低溫度設定
3. 門窗未關閉 → 關閉門窗
4. 室外機散熱不良 → 清潔室外機
5. 冷媒不足 → 聯絡專業維修補充冷媒
6. 房間太大超過冷氣能力 → 考慮更換較大機型

問題: 運轉有異音
可能原因:
1. 濾網未裝好 → 重新安裝濾網
2. 室外機固定不穩 → 檢查固定螺絲
3. 葉片損壞 → 更換葉片
4. 風扇故障 → 聯絡維修

問題: 漏水
可能原因:
1. 排水管堵塞 → 清理排水管
2. 排水管斜度不足 → 調整斜度
3. 排水管破裂 → 更換排水管
4. 結露水過多 → 增加保溫材

4.2 錯誤代碼說明

E1 - 室內溫度感應器異常
處理: 關閉電源5分鐘後重試,若持續出現請聯絡維修

E2 - 室外溫度感應器異常
處理: 檢查室外機,聯絡維修

E3 - 壓縮機過載保護
處理: 關閉30分鐘後重試,檢查室外機散熱

E4 - 系統高壓保護
處理: 清潔室外機,確保散熱良好

E5 - 室內風扇馬達異常
處理: 檢查濾網安裝,聯絡維修
