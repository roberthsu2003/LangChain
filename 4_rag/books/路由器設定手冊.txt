第一章：硬體規格與特色
========================================

1.1 產品規格
----------------------------------------
型號：TR-5000
無線標準：IEEE 802.11ax (WiFi 6)
傳輸速率：
- 2.4GHz：最高 574 Mbps
- 5GHz：最高 2402 Mbps
- 總和速率：2976 Mbps (AX3000)

處理器：1.5GHz 四核心處理器
記憶體：512MB RAM / 256MB Flash
天線：外接式 4根（支援波束成型）
網路介面：
- 1個 WAN埠（10/100/1000 Mbps）
- 4個 LAN埠（10/100/1000 Mbps）
- 1個 USB 3.0埠

支援裝置數：最多 128個裝置同時連線
訊號範圍：約 150坪（開放空間）
電源規格：DC 12V / 2A
尺寸：260mm × 160mm × 45mm
重量：450公克

1.2 主要功能特色
----------------------------------------
WiFi 6 技術：
- OFDMA技術（提升多裝置效率）
- MU-MIMO（多用戶同時傳輸）
- BSS Coloring（降低干擾）
- Target Wake Time（省電功能）

安全防護：
- WPA3加密（最新安全標準）
- 防火牆保護
- DoS攻擊防禦
- 家長控制功能
- VPN伺服器支援

智慧管理：
- 手機 App遠端管理
- 自動韌體更新
- QoS頻寬管理
- 訪客網路獨立
- 網路診斷工具

1.3 包裝內容物
----------------------------------------
標準配件：
□ TR-5000 路由器主機 × 1
□ 電源變壓器 × 1
□ 網路線（1.5公尺）× 1
□ 快速安裝指南 × 1
□ 保固卡 × 1

初次使用需準備：
- 寬頻數據機（小烏龜）
- 電腦或智慧型手機
- ISP提供的帳號密碼（PPPOE撥接）

----------------------------------------
前面板指示燈（由左至右）：

5G WiFi燈：

LAN燈（1-4）：
- 綠燈恆亮：有裝置連接
- 綠燈閃爍：資料傳輸中
- 熄滅：無裝置連接

========================================
第二章：初次設定
========================================

2.1 硬體連接
----------------------------------------
步驟 1：放置路由器
理想位置：
- 房屋中央位置（訊號覆蓋最佳）
- 高度約 1-2公尺
- 遠離金屬物品與電器（減少干擾）
- 避免密閉空間（如櫃子內）
- 保持通風散熱

步驟 2：連接網路線
1. 關閉數據機（小烏龜）電源
2. 將數據機的網路線接至路由器「藍色 WAN埠」
3. 確認接頭插緊（聽到「喀」一聲）
4. 若要使用有線網路，將電腦接至「黃色 LAN埠」

步驟 3：接上電源
1. 將電源變壓器接至路由器
2. 插上電源插座
3. 按下路由器背面電源開關
4. 等待約 60-90秒啟動完成

步驟 4：開啟數據機
1. 開啟數據機電源
2. 等待數據機完全啟動（約 2分鐘）
3. 確認數據機指示燈正常

啟動完成確認：
□ 電源燈恆亮綠燈
□ Internet燈亮起（綠或橘）
□ WiFi燈恆亮綠燈
□ 可搜尋到預設 WiFi訊號

2.2 使用電腦進行初次設定
----------------------------------------
步驟 1：連接路由器
方式 A：有線連接
- 使用網路線連接電腦與路由器 LAN埠

方式 B：無線連接
- 搜尋預設 WiFi名稱（標籤在路由器底部）
- 預設格式：TR-5000_XXXX
- 輸入預設密碼（標籤上的 8位數字）

步驟 2：開啟設定頁面
1. 開啟瀏覽器（Chrome、Edge、Safari皆可）
2. 輸入網址：http://192.168.1.1
3. 或輸入：http://router.setup
4. 按 Enter鍵

步驟 3：登入管理介面
- 預設帳號：admin
- 預設密碼：admin
- 首次登入會要求更改密碼

步驟 4：快速設定精靈
系統會自動啟動設定精靈：

1. 設定管理員密碼
   - 建議使用 8-16位英數混合
   - 包含大小寫字母與數字
   - 記得妥善保管密碼

2. 選擇時區
   - 選擇「台北（GMT+8）」
   - 系統將自動同步網路時間

3. 網際網路連線設定
   根據 ISP業者選擇：

   選項 A：PPPOE撥接（中華電信、台灣大哥大等）
   - 輸入 ISP提供的帳號（通常是電話號碼@hinet.net）
   - 輸入密碼
   - 點選「連線」測試

   選項 B：動態 IP（部分社區網路、Cable數據機）
   - 選擇「自動取得 IP」
   - 系統自動設定

   選項 C：固定 IP（企業用戶）
   - 輸入 ISP提供的 IP位址
   - 輸入子網路遮罩
   - 輸入預設閘道
   - 輸入 DNS伺服器

4. 完成設定
   - 點選「完成」
   - 路由器將重新啟動（約 30秒）
   - 重新連線後即可上網

2.3 使用手機 App設定
----------------------------------------
步驟 1：下載 App
- iOS：至 App Store搜尋「TR Router」
- Android：至 Play商店搜尋「TR Router」
- 或掃描說明書上的 QR Code

步驟 2：連接路由器 WiFi
- 開啟手機 WiFi設定
- 連接預設 WiFi：TR-5000_XXXX
- 輸入預設密碼（路由器底部標籤）

步驟 3：開啟 App設定
1. 啟動 TR Router App
2. App會自動偵測路由器
3. 點選「開始設定」
4. 依畫面指示完成設定

步驟 4：設定內容
與電腦版相同：
- 建立管理員密碼
- 設定網際網路連線
- 完成後可立即使用

優點：
- 操作介面更友善
- 支援遠端管理
- 即時通知功能
- 輕鬆管理連線裝置

========================================
第三章：WiFi 無線網路設定
========================================

3.1 WiFi 基本設定
----------------------------------------
進入 WiFi設定：
1. 登入管理介面（192.168.1.1）
2. 點選「無線網路」→「基本設定」

2.4GHz 頻段設定：
網路名稱（SSID）：
- 建議使用易記且安全的名稱
- 避免使用個人資訊（如地址、電話）
- 支援中文但部分裝置可能無法顯示
- 範例：Home_2.4G、MyHome_24

安全性設定：
- 加密方式：選擇「WPA3-Personal」（最安全）
- 若有舊裝置不支援，選「WPA2/WPA3 混合模式」
- 密碼長度：8-63字元
- 建議密碼格式：英文大小寫+數字+符號

頻道設定：
- 建議選擇「自動」
- 手動選擇：1、6、11（較少干擾）
- 頻道寬度：20/40MHz 自動

5GHz 頻段設定：
網路名稱（SSID）：
- 建議與 2.4G區分（如加上_5G）
- 範例：Home_5G、MyHome_5G

頻道設定：
- 建議選擇「自動」
- 手動選擇：36、40、44、48（DFS頻段需注意）
- 頻道寬度：20/40/80MHz 自動

進階選項：
□ 啟用無線網路：勾選（預設開啟）
□ 隱藏 SSID：不建議勾選（會影響連線便利性）
□ 啟用 WiFi 6：勾選（提升效能）
□ 啟用波束成型：勾選（增強訊號）

3.2 雙頻合一（Smart Connect）
----------------------------------------
功能說明：
- 將 2.4G與 5G合併為同一個 WiFi名稱
- 路由器自動為裝置選擇最佳頻段
- 簡化連線步驟

設定方法：
1. 「無線網路」→「Smart Connect」
2. 勾選「啟用 Smart Connect」
3. 設定統一的 WiFi名稱與密碼
4. 儲存設定

優點：
- 裝置自動切換最佳頻段
- 單一密碼方便管理
- 無縫漫遊體驗

缺點：
- 無法手動指定裝置使用哪個頻段
- 部分舊裝置可能連線不穩

建議使用情境：
- 家中裝置都支援雙頻
- 不需特別指定頻段
- 追求便利性

3.3 訪客網路設定
----------------------------------------
功能說明：
- 獨立的 WiFi網路給訪客使用
- 訪客無法存取家中內部網路裝置
- 保護隱私與安全

設定步驟：
1. 「無線網路」→「訪客網路」
2. 勾選「啟用訪客網路」

2.4G訪客網路：
- SSID名稱：Guest_2.4G（可自訂）
- 安全性：WPA2-Personal
- 密碼：設定簡單好記的密碼
- 啟用存取隔離：勾選（訪客間無法互連）

5G訪客網路：
- 可同時啟用或單獨使用
- 設定方式同 2.4G

進階選項：
□ 限制頻寬：可限制訪客使用頻寬（如 10Mbps）
□ 限制使用時間：設定特定時段開放
□ 最大連線數：限制同時連線裝置數量

使用時機：
- 朋友來訪需要 WiFi
- Airbnb出租房客使用
- 商家提供客人上網
- 臨時性網路分享

3.4 WiFi 排程
----------------------------------------
功能說明：
- 設定 WiFi開關的時間排程
- 省電且控制使用時段
- 可分別設定主網路與訪客網路

設定步驟：
1. 「無線網路」→「WiFi排程」
2. 勾選「啟用 WiFi排程」
3. 設定關閉時段

範例設定：
工作日排程（週一～週五）：
- 00:00 - 07:00：關閉（睡眠時段）
- 07:00 - 08:30：開啟（早晨使用）
- 08:30 - 17:30：關閉（上班時段，可選）
- 17:30 - 24:00：開啟（晚間使用）

週末排程（週六～週日）：
- 全天開啟（或依需求設定）

兒童上網管理：
- 22:00 - 07:00：關閉（確保睡眠）
- 可搭配家長控制功能

注意事項：
- 排程關閉期間無法連線
- 不影響有線網路
- 可隨時手動開關

========================================
第四章：密碼更改與安全設定
========================================

4.1 更改管理員密碼
----------------------------------------
重要性：
- 預設密碼容易被猜測
- 保護路由器設定不被竄改
- 防止未授權存取

更改步驟：
1. 登入管理介面
2. 「系統設定」→「密碼管理」
3. 輸入目前密碼
4. 輸入新密碼（兩次確認）
5. 點選「套用」

密碼建議：
- 長度：至少 10位字元
- 組成：大小寫英文+數字+符號
- 範例：MyR0uter@2024!
- 避免使用：生日、電話、123456

安全提醒：
□ 定期更換密碼（建議每 6個月）
□ 勿使用與 WiFi相同密碼
□ 妥善保管，可用密碼管理軟體
□ 勿分享給他人

4.2 更改 WiFi 密碼
----------------------------------------
更改時機：
- 懷疑密碼外洩
- 定期安全更新
- 訪客知道密碼後
- 搬家或出租房屋

更改步驟：
1. 「無線網路」→「安全性設定」
2. 選擇要更改的頻段（2.4G / 5G）
3. 在「密碼」欄位輸入新密碼
4. 點選「套用」
5. 重新啟動路由器

密碼強度建議：
弱密碼（不建議）：
- 12345678
- password
- 88888888

中等密碼（尚可）：
- myhome2024
- wifi@home

強密碼（建議）：
- MyHome@Wifi2024!
- Secure#Home$5G

超強密碼（最佳）：
- 2Bor!2B@That$5G
- Tr5k#Wifi*2024

4.3 MAC位址過濾
----------------------------------------
功能說明：
- 只允許特定裝置連線
- 每個網路裝置都有唯一的 MAC位址
- 提供額外安全層級

設定步驟：
1. 「無線網路」→「MAC位址過濾」
2. 選擇模式：

模式一：黑名單模式
- 封鎖特定 MAC位址
- 其他裝置可自由連線
- 適合：封鎖特定裝置

模式二：白名單模式（更安全）
- 只允許清單內的 MAC位址連線
- 其他裝置一律拒絕
- 適合：高安全需求環境

3. 新增 MAC位址
方法 A：手動輸入
- 格式：AA:BB:CC:DD:EE:FF
- 備註：填寫裝置名稱（如：小明的手機）

方法 B：從連線清單選擇
- 查看目前連線裝置
- 直接勾選加入清單

查詢裝置 MAC位址：
手機（iOS）：
設定 → 一般 → 關於本機 → WiFi位址

手機（Android）：
設定 → 關於手機 → 狀態 → WiFi MAC位址

電腦（Windows）：
命令提示字元 → 輸入「ipconfig /all」

電腦（Mac）：
系統偏好設定 → 網路 → 進階 → WiFi → MAC位址

4.4 防火牆設定
----------------------------------------
基本防火牆：
1. 「安全性」→「防火牆」
2. 啟用 SPI防火牆（預設開啟）
3. 功能：阻擋可疑的進入連線

DoS攻擊防護：
- 自動偵測並阻擋 DoS攻擊
- 保護網路不被癱瘓
- 建議開啟

進階設定：
□ 封鎖 WAN端 Ping（防止被掃描）
□ 啟用 ALG（應用程式層級閘道）
□ 啟用 UPnP（網路遊戲、視訊通話需要）

通訊埠轉發（Port Forwarding）：
用途：讓外部網路存取內部裝置
範例應用：
- 架設網站伺服器
- 遠端監控攝影機
- 遠端桌面連線
- NAS外部存取

設定方法：
1. 「進階設定」→「通訊埠轉發」
2. 填寫：
   - 服務名稱：自訂（如：Web Server）
   - 外部埠：8080
   - 內部 IP：192.168.1.100
   - 內部埠：80
   - 通訊協定：TCP
3. 儲存設定

安全提醒：
- 非必要勿開放通訊埠
- 開放後定期檢查
- 使用非標準埠號（如 8080取代 80）
- 搭配強密碼與加密

========================================
第五章：家長控制功能
========================================

5.1 網站過濾
----------------------------------------
功能說明：
- 限制特定裝置存取特定網站
- 保護兒童遠離不當內容
- 提升工作效率（封鎖娛樂網站）

設定步驟：
1. 「家長控制」→「網站過濾」
2. 啟用網站過濾功能
3. 選擇過濾模式

模式一：黑名單模式
- 封鎖特定網站
- 其他網站可正常瀏覽
- 範例：封鎖 facebook.com、youtube.com

模式二：白名單模式（更嚴格）
- 只允許清單內網站
- 其他網站一律封鎖
- 適合年幼兒童

4. 新增網址
   - 輸入網域名稱（如：example.com）
   - 支援關鍵字過濾（如：game、video）
   - 可匯入分類清單（賭博、色情等）

5. 選擇套用的裝置
   - 可依 MAC位址指定
   - 或套用至所有裝置

常用封鎖清單範例：
兒童保護：
- 成人內容網站
- 賭博網站
- 暴力遊戲網站

工作效率：
- facebook.com
- instagram.com
- youtube.com（或僅限時段）
- 線上遊戲網站

5.2 上網時間管理
----------------------------------------
功能說明：
- 限制特定裝置的上網時段
- 培養良好作息習慣
- 平衡學習與娛樂

設定步驟：
1. 「家長控制」→「時間管理」
2. 新增規則
3. 選擇裝置（依 MAC位址或 IP）
4. 設定允許上網時段

範例設定一：學齡兒童
平日（週一～週五）：
- 07:00-08:00：允許（早晨）
- 08:00-15:00：封鎖（上課時間）
- 15:00-18:00：允許（作業查資料）
- 18:00-19:00：封鎖（晚餐時間）
- 19:00-21:00：允許（2小時娛樂）
- 21:00-07:00：封鎖（就寢時間）

週末（週六～週日）：
- 08:00-22:00：允許
- 22:00-08:00：封鎖

範例設定二：青少年
- 較寬鬆的時間限制
- 僅限制深夜時段（23:00-07:00）
- 考試期間可臨時調整

範例設定三：工作電腦
- 上班時間封鎖娛樂網站
- 午休時段（12:00-13:00）開放
- 下班後全面開放

5.3 應用程式管理
----------------------------------------
功能說明：
- 限制特定類型應用程式
- 管理線上遊戲使用
- 控制 P2P下載

設定方式：
1. 「家長控制」→「應用程式控制」
2. 選擇要限制的類別

可限制類型：
□ 線上遊戲（如：英雄聯盟、傳說對決）
□ 串流影音（如：Netflix、YouTube）
□ 社群軟體（如：Line、Facebook）
□ P2P下載（如：BitTorrent）
□ 即時通訊（如：Messenger、Telegram）

進階設定：
- 可設定允許時段
- 可限制頻寬使用
- 可針對特定裝置套用

5.4 活動報告
----------------------------------------
功能說明：
- 查看裝置上網紀錄
- 了解網路使用狀況

第六章：連線問題排除
========================================

6.1 無法連線至網際網路
----------------------------------------
症狀：
- WiFi可連線但無法上網
- 網頁無法開啟
- Internet燈號顯示橘燈或不亮

檢查步驟：

步驟 1：確認線路連接
□ 數據機電源是否開啟
□ 網路線是否鬆脫
□ WAN埠連接是否正確（藍色埠）
□ 數據機指示燈是否正常

步驟 2：重新啟動裝置
1. 關閉路由器電源
2. 關閉數據機電源
3. 等待 30秒
4. 先開啟數據機，等待 2分鐘
5. 再開啟路由器，等待 1分鐘
6. 測試連線

步驟 3：檢查 WAN設定
1. 登入管理介面
2. 「網際網路」→「連線設定」
3. 確認連線類型正確
4. 若為 PPPOE，檢查帳號密碼
5. 點選「重新連線」

步驟 4：確認 ISP帳號
- 聯絡 ISP業者確認帳號狀態
- 確認是否欠費停用
- 確認服務區域是否維修

步驟 5：更換網路線
- 使用其他網路線測試
- 確認線材品質良好

步驟 6：韌體更新
- 檢查是否有新版韌體
- 更新至最新版本

6.2 WiFi 訊號不穩定
----------------------------------------
症狀：
- 訊號時強時弱
- 連線經常中斷
- 特定區域收不到訊號

解決方法：

問題一：訊號干擾
可能來源：
- 微波爐（使用時避免）
- 無線電話（2.4GHz）
- 藍牙裝置
- 鄰居的 WiFi

解決方式：
1. 更換 WiFi頻道
   - 2.4GHz：改用 1、6、11
   - 5GHz：改用 36、40、149、153
2. 改用 5GHz頻段（較少干擾）
3. 調整路由器位置，遠離干擾源

問題二：距離過遠
症狀：
- 離路由器遠就斷線
- 訊號格數很少

解決方式：
1. 調整路由器位置至中央
2. 提高路由器擺放高度
3. 調整天線角度（建議 2支垂直、2支 45度）
4. 加裝 WiFi延伸器（Extender）
5. 升級為 Mesh網狀網路系統

問題三：連線裝置過多
症狀：
- 多人使用時變慢
- 尖峰時段特別卡

解決方式：
1. 啟用 QoS頻寬管理
2. 限制部分裝置頻寬
3. 使用 5GHz分流裝置
4. 升級至更高階路由器

問題四：牆壁阻隔
不同材質影響：
- 木板：訊號衰減約 10%
- 石膏板：訊號衰減約 20%
- 磚牆：訊號衰減約 40%
- 水泥牆：訊號衰減約 60%
- 金屬：幾乎完全阻隔

解決方式：
- 避免多層牆壁阻隔
- 考慮使用有線網路
- 加裝 WiFi延伸器

6.3 連線速度緩慢
----------------------------------------
檢查步驟：

步驟 1：測試實際速度
使用測速網站：
- Speedtest.net
- Fast.com
- 中華電信測速網站

測試方式：
- 使用有線連接測試
- 單一裝置測試
- 關閉其他裝置
- 記錄下載/上傳速度

步驟 2：比對合約速度
- 確認 ISP提供的速率
- 實測應達合約速度的 80%以上
- 若差距過大，聯絡 ISP檢查

步驟 3：檢查路由器 CPU使用率
1. 管理介面 →「系統狀態」
2. 查看 CPU使用率
3. 若長期超過 80%，可能需要升級路由器

步驟 4：檢查連線裝置
- 查看是否有裝置大量下載
- 檢查是否有未知裝置連線（可能被盜用）
- 更改 WiFi密碼

步驟 5：啟用 QoS
1. 「進階設定」→「QoS」
2. 設定頻寬優先順序
3. 重要應用（視訊會議、遊戲）優先

步驟 6：更新韌體
- 新版韌體可能改善效能
- 定期檢查更新

6.4 裝置無法連線
----------------------------------------
症狀：
- 輸入密碼後無法連線
- 顯示「無法取得 IP位址」
- 一直顯示「連線中」

解決方法：

問題一：密碼錯誤
- 重新確認密碼（注意大小寫）
- 重設 WiFi密碼後重試
- 避免特殊符號（部分舊裝置不支援）

問題二：IP位址衝突
解決步驟：
1. 管理介面 →「DHCP設定」
2. 重新啟動 DHCP伺服器
3. 或將裝置設為靜態 IP

問題三：MAC位址被封鎖
檢查方式：
1. 「MAC位址過濾」
2. 確認裝置不在黑名單
3. 或加入白名單

問題四：達到連線上限
- 檢查目前連線裝置數量
- 踢除閒置裝置
- 提高連線上限設定

問題五：裝置端問題
手機/平板：
- 忘記網路後重新連線
- 重新啟動裝置
- 更新系統版本

電腦：
- 更新網路卡驅動程式
- 重設網路設定（Windows：網路重設）
- 檢查防火牆設定

6.5 無法進入管理介面
----------------------------------------
症狀：
- 無法開啟 192.168.1.1
- 瀏覽器顯示無法連線

解決方法：

步驟 1：確認連線
- 確認已連上路由器（WiFi或有線）
- 關閉行動網路（手機）

步驟 2：確認 IP位址
1. 查詢電腦取得的閘道位址
   Windows：cmd → ipconfig
   Mac：終端機 → netstat -nr | grep default
2. 使用正確的閘道 IP進入

步驟 3：清除瀏覽器快取
- 清除瀏覽器 Cookie與快取
- 或使用無痕模式開啟

步驟 4：更換瀏覽器
- 嘗試其他瀏覽器
- 避免使用過舊版本

步驟 5：重置路由器
最後手段（將恢復原廠設定）：
1. 找到路由器背面 Reset孔
2. 使用迴紋針按住 10秒
3. 指示燈全亮後放開
4. 等待重新啟動
5. 使用預設帳密登入

----------------------------------------

- 新增功能

附錄：常見問題 FAQ
========================================

Q1：忘記管理員密碼怎麼辦？
A1：只能重置路由器恢復原廠設定，所有設定將清除。

Q2：WiFi訊號要怎麼加強？
A2：(1)調整路由器位置 (2)更換頻道 (3)加裝延伸器 (4)升級天線

Q3：2.4G和 5G有什麼差別？
A3：
- 2.4G：範圍較廣、穿牆較好、速度較慢、干擾較多
- 5G：範圍較小、穿牆較差、速度較快、干擾較少

Q4：要選擇哪個 WiFi頻段？
A4：
- 距離遠、隔牆多 → 2.4G
- 同一房間、需要高速 → 5G
- 建議啟用雙頻合一，自動選擇

Q5：路由器需要關機嗎？
A5：不需要，但建議每月重新啟動一次，清除暫存記憶體。

Q6：幾年需要更換路由器？
A6：一般 3-5年，或當：
- 韌體不再更新
- 速度無法滿足需求
- 經常當機
- 不支援新技術（如 WiFi 6）
